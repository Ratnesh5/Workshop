# AI & Robotics Summer Workshop

A responsive workshop landing page built using React.js, Tailwind CSS, Express.js, and MongoDB Atlas.

## Features

- Responsive UI
- React Component Architecture
- Form Validation
- Express API
- MongoDB Integration
- Vercel + Render Deployment

## Tech Stack

Frontend:
- React.js
- Tailwind CSS
- Axios
- React Icons

Backend:
- Node.js
- Express.js
- MongoDB Atlas
- Mongoose

## Deployment

Frontend: Vercel
Backend: Render
